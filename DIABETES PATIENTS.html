In X all the independent variables are stored In Y the predictor variable(OUTCOME) is stored. Train-test split is a technique used in machine learning to assess model performance. It divides the dataset into a training set and a testing set, with a 0.2 test size indicating that 20% of the data is used for testing and 80% for training.
